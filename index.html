Survey

Application for the management of questionnaires

View project on GitHub

This application can create, edit and answer questionnaires.

Parts

  • XML database with questions and responses
  • desktop interface for creating and editing questions
  • web interface for rendering and answering questions

Details of the Survey and how to run it are on the wiki. Javadoc API is here.

There is also Gitter chat room for talking about this application.
